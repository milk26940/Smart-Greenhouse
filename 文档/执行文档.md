按这个顺序在开发板上运行就行。

先登录开发板：

```bash
ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedAlgorithms=+ssh-rsa root@192.168.137.106
```

确认 NFS 已挂载：

```bash
mount | grep /mnt/nfs
```

如果没有挂载，执行：

```bash
mkdir -p /mnt/nfs
mount -t nfs -o vers=3,nolock,proto=tcp 192.168.137.3:/home/lan/linux/nfs /mnt/nfs
```

进入英文目录并确认程序存在：

```bash
cd /mnt/nfs/smart-greenhouse/mx6u
ls -l build/mx6u_skeleton
```

直接运行：

```bash
./build/mx6u_skeleton
```

如果你想先降低负载再运行，可以关掉 MQTT：

```bash
export MX6U_MQTT_ENABLE=0
./build/mx6u_skeleton
```

如果要限时运行 20 秒做测试：

```bash
timeout 20 ./build/mx6u_skeleton
```

运行后你会看到这些内容：

- 串口收到的 `ENV` 数据
- `STATE SNAPSHOT`
- `VISION SNAPSHOT`
- `DECISION SNAPSHOT`
- `MQTT SNAPSHOT`

结束程序用：

```bash
Ctrl + C
```

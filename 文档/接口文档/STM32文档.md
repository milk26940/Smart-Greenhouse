# STM32接线文档

## 1. 模块接线总表

| 模块 | 信号 | STM32 引脚 | 说明 |
| --- | --- | --- | --- |
| LoRa 模块 | TX | PA3 / USART2_RX | LoRa 发给 STM32 |
| LoRa 模块 | RX | PA2 / USART2_TX | STM32 发给 LoRa |
| LoRa 模块 | AUX | PC13 | 当前工程已有 |
| DHT11 | DATA | PB12 | 单总线，建议外接上拉 |
| 光敏模块 | AO | PC0 / ADC1_IN10 | 模拟量输入 |
| CO2 串口模块 | TX | PB11 / USART3_RX | 传感器发给 STM32 |
| CO2 串口模块 | RX | PB10 / USART3_TX | 若模块支持回传/配置可接 |
| 电机驱动 | PWM | PB0 / TIM3_CH3 | 速度控制 |
| 电机驱动 | EN | PB1 | 使能控制 |
| 电机驱动 | DIR | PB5 | 当前实现固定正向，启用时拉高 |
| 调试串口到 PC | TX | PA9 / USART1_TX | 调试输出 |
| 调试串口到 PC | RX | PA10 / USART1_RX | 保留 |

## 2. 供电说明

- STM32 主控：按开发板原有供电
- DHT11：通常 `3.3V` 或 `5V`，优先确认模块版本
- 光敏模块：若输出接 `PC0`，必须保证模拟输出不超过 `3.3V`
- CO2 串口模块：确认串口电平为 `3.3V TTL`
- 电机驱动：电机电源与逻辑电源分开，逻辑地必须和 STM32 共地

## 3. 关键注意事项

- DHT11 数据线若模块板上没有上拉，需补 `4.7k~10k` 上拉到 `3.3V`
- 光敏若使用分压电路，确认 ADC 输入范围安全
- CO2 模块如果是 `5V TTL`，先做电平转换再接 `PB11`
- 电机驱动不要直接接电机到 STM32，引脚只接驱动器控制端
- 电机启动瞬间干扰较大，建议电机电源侧增加去耦和续流保护

## 4. 当前软件约定

- 电机协议当前只开放启停和速度，不开放反转控制
- `PB5` 已预留为方向脚，当前软件中启用电机时固定置高
- LoRa 上报周期固定 5 秒
- `USART1 -> PC` 调试转发逻辑保留不变
